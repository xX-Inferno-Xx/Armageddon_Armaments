# Armageddon_Armaments
My hobby project mod for Starsector. Some ships, some weapons, a new faction.

A mod to introduce a faction dedicated to defending the sector

Armageddon Arms, being a PMC, will sometimes sell their ships to other factions. This, however, can be configured by playing around in data/world/factions.
If you wanna control what ships factions get, how often they get them, or even what factions can get ships, then poke around in those files.

Featuring:
- Powerful warships
- A new faction
- Custom portraits
- Some neat lil' main menu missions
